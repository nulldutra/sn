# Welcome to sn

sn is a TUI notes app inspired by Obsidian.

## How to use

- **a**: create a new note (opens a name prompt)
- **i**: edit the selected note in the preview panel
- **Esc**: save changes and leave edit mode
- **j/k** or **↑/↓**: navigate the browser list
- **h/l** or **←/→**: go to parent / enter folder
- **Enter**: enter folder or go up from `../`
- **[** / **]**: scroll note content
- **g** / **G**: jump to the first / last item
- **q**: quit

## Organization

Place `.md` or `.txt` files in the `notes/` directory.
Use subfolders to organize notes — open folders with **l** or **Enter**.
The preview panel renders Markdown formatting automatically.
